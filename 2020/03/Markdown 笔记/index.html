Markdown 笔记

发表于2020-03-02|更新于2022-03-11|笔记Markdown
|字数总计:578|阅读时长:1分钟|阅读量:

Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

标题

1级标题

使用=（一个以上）或者#（一个）+ 空格

2级标题

使用-（一个以上）或者#（两个）+ 空格

3-6级标题

使用#（3-6个）+ 空格

段落

换行

使用两个以上空格加回车

字体

  1. 斜体
    使用一对*或者_
  2. 粗体
    使用一对**或者__
  3. 粗斜体
    使用一对***或者___

分割线

使用两个以上的*、-或者_

删除线

使用波浪线~~删除线文本~~

下划线

使用HTML的标签<u>下划线文本</u>

脚注

使用[^脚注] +[^脚注]:脚注内容

角标

  1. 上角标：使用x^2^–x^2^
  2. 下角标：使用H~2~O–H2O

高亮

使用等号==高亮文本==

居中

使用HTML的标签<center>居中文本</center>

目录

使用[TOC]

表情

使用:单词:

列表

可分为有序列表和无序列表，可嵌套使用

有序列表

使用数字 + . + 空格

无序列表

使用*、+或者- + 空格

区块

使用> + 空格，可嵌套，并可与列表嵌套

代码

一行代码

使用一对反引号

一段代码

使用三个反引号

链接

  1. 使用[链接名称](链接地址)（链接名称可省略）
  2. 使用[链接名称][变量]（与脚注类似）

图片

  1. 使用![alt 属性文本](图片地址)
  2. 把图片直接拖入文档自动创建

表格

  1. 使用|来分隔单元格
  2. 使用-来分隔表头
  3. 对齐方式
    • 左对齐：:---
    • 右对齐：---:
    • 居中对齐：:---:

附：快捷键

快捷键 功能 快捷键 功能
Ctrl+1 一阶标题 Ctrl+B 字体加粗
Ctrl+2 二阶标题 Ctrl+I 字体倾斜
Ctrl+3 三阶标题 Ctrl+U 下划线
Ctrl+4 四阶标题 Ctrl+Home 返回顶部
Ctrl+5 五阶标题 Ctrl+End 返回底部
Ctrl+6 六阶标题 Ctrl+T 创建表格
Ctrl+L 选中某句话 Ctrl+K 创建超链接
Ctrl+D 选中某个单词 Ctrl+F 搜索
Ctrl+E 选中相同格式的文字 Ctrl+H 搜索并替换
Alt+Shift+5 删除线 Ctrl+Shift+I 插入图片
Ctrl+Shift+M 公式块 Ctrl+Shift+Q 引用
